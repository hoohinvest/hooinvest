## Field Descriptions

### Raises
- `id`: Unique raise identifier (UUID)
- `businessId`: Business owner identifier
- `title`: Campaign title
- `description`: Campaign description (optional)
- `goalCents`: Funding target in cents (e.g., 100000 = $1000)
- `minContributionCents`: Minimum investment amount in cents
- `maxContributionCents`: Maximum investment per investor in cents (optional)
- `instrument`: EQUITY | INTEREST | ROYALTY
- `instrumentTermsJson`: JSON string with instrument-specific terms
- `status`: DRAFT | OPEN | FUNDED | CLOSED | CANCELLED | EXPIRED | REFUNDING
- `raisedCents`: Current funding amount in cents
- `expiresAt`: ISO timestamp when raise expires
- `createdAt`: ISO timestamp when raise was created
- `updatedAt`: ISO timestamp when raise was last updated

### Investments
- `id`: Unique investment identifier (UUID)
- `raiseId`: Associated raise identifier
- `investorId`: Investor identifier
- `amountCents`: Investment amount in cents
- `status`: PENDING | AUTHORIZED | CAPTURED | REFUNDED | FAILED
- `paymentProviderId`: Stripe PaymentIntent identifier
- `createdAt`: ISO timestamp when investment was created
- `updatedAt`: ISO timestamp when investment was last updated

### Allocations
- `id`: Unique allocation identifier (UUID)
- `raiseId`: Associated raise identifier
- `investorId`: Investor identifier
- `allocationJson`: JSON string with allocation details
- `certificateNumber`: Unique certificate identifier (e.g., HOO-12345678-87654321-ABC123)
- `createdAt`: ISO timestamp when allocation was created

### Payouts
- `id`: Unique payout identifier (UUID)
- `raiseId`: Associated raise identifier
- `amountCents`: Net payout amount in cents (after platform fees)
- `status`: PENDING | RELEASED | FAILED
- `providerTransferId`: Stripe transfer identifier (optional)
- `createdAt`: ISO timestamp when payout was created
- `updatedAt`: ISO timestamp when payout was last updated

### Audit Logs
- `id`: Unique audit log identifier (UUID)
- `type`: Event type (e.g., RAISE_CREATED, PAYMENT_SUCCEEDED, ALLOCATION_COMPLETED)
- `payloadJson`: JSON string with event details
- `createdAt`: ISO timestamp when event occurred
